## Description

This is a frontend prototype for **Heartfelt Hospitality**, a leading hospitality consulting company specializing in hotel development, operations, and asset management across India and beyond. The prototype showcases the website's design and functionality, built as a static frontend application using PHP for templating.

**Note:** This is a prototype version of the frontend. It demonstrates the user interface and basic interactions but does not include backend functionality or dynamic data processing.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices using Bootstrap framework.
- **Interactive Carousel**: Dynamic image carousel on the homepage featuring key services and messages.
- **Service Pages**: Dedicated pages for various hospitality services including:
  - Concept Planning
  - Feasibility Study
  - Pre-Opening Support
  - Operations Management
  - Hotel Asset Management
  - Technical Assistance
- **Team Showcase**: Information about the company's team and expertise.
- **Contact & FAQs**: Contact forms and frequently asked questions section.
- **SEO Optimized**: Includes meta tags for better search engine visibility.
- **Modern UI**: Clean, professional design with animations and smooth transitions.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Framework**: Bootstrap 4.4.1
- **Libraries**:
  - Owl Carousel for image sliders
  - Animate.css for animations
  - Font Awesome for icons
  - Waypoints and CounterUp for interactive elements
- **Backend (Templating)**: PHP
- **Fonts**: Google Fonts (Lato, Oswald)

## Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-username/heartfelt-hospitality-prototype.git
   cd heartfelt-hospitality-prototype
   ```

2. **Set Up Local Server**:
   - Ensure you have a local web server with PHP support (e.g., Apache with PHP, or use XAMPP/WAMP).
   - Place the project files in your web server's root directory (e.g., `htdocs` for XAMPP).

3. **Open in Browser**:
   - Navigate to `http://localhost/your-project-folder/index.php` in your web browser.

## Usage

- **Homepage**: Access `index.php` to view the main landing page with carousel and overview.
- **Navigation**: Use the header navigation to explore different service pages and sections.
- **Contact**: Fill out the contact form on `contact.php` for inquiries.
- **Responsive Testing**: Resize your browser or use developer tools to test mobile responsiveness.

This prototype is designed for demonstration purposes and can be viewed directly in a web browser without additional setup beyond a basic web server.
